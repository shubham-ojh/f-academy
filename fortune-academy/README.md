# Fortune Academy – Website

A professional, high-conversion website for **Fortune Academy**, a premier coaching institute. Design: Deep Navy Blue, Gold, and White.

## What’s included

- **Hero:** “Your Success, Our Mission” + sub-headline (Class 5–10 Boards, 11–12, IIT-JEE, KCET, NDA) + “Book a Free Demo” CTA  
- **Courses/Programs:** Cards for Foundations (5–10), Pre-University (11–12 Science), Competitive Excellence (IIT-JEE, KCET, NDA)  
- **Why Choose Us:** Experienced Faculty, Personalized Mentorship, Result-Oriented Curriculum  
- **Faculty profiles:** Short bios (e.g. Ex-IITian, 15+ years)  
- **Results/Toppers gallery:** Scrollable carousel of student achievements  
- **NDA Special Wing:** Dedicated section with NDA offerings  
- **Inquiry form:** Parent name, phone, student class/exam, message – “Request Callback”  
- **Navigation:** Home | Courses | Results | NDA Special Wing | Contact Us  
- **Sticky CTA:** Call Now + WhatsApp buttons fixed at bottom-right (mobile-friendly)

## Run locally

Open `index.html` in a browser, or:

```bash
cd fortune-academy
npx serve .
```

## Logo

Place your Fortune Academy logo in `assets/logo.png` (nav and hero use it). If the file is missing, the site still shows the text “Fortune Academy” in the nav.

## Customize

- **Phone/WhatsApp:** In `index.html`, replace `+919876543210` and `919876543210` in the sticky CTA links with your real number.  
- **Faculty/Toppers:** Edit the names, qualifications, and results in the HTML.  
- **Syllabus text:** Add detailed descriptions for NDA, KCET, JEE in the course cards or new sections if needed.
